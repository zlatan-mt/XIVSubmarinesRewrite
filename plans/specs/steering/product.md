# プロダクト概要 - XIV Submarines Rewrite

## プロダクト名
XIV Submarines Rewrite

## プロダクトビジョン
Final Fantasy XIV の Dalamud プラグインとして、潜水艦探索の管理を自動化し、プレイヤーの時間を節約する。Discord や Notion への通知機能により、ゲーム外でも帰港状況を把握できる次世代の潜水艦管理ツール。

## ターゲットユーザー
- Final Fantasy XIV のプレイヤー（特に工房潜水艦を活用するプレイヤー）
- 複数キャラクターで潜水艦を運用している中～上級者
- Discord や Notion で情報管理を行っているプレイヤー

## 主要機能

### 1. 自動データ取得
- 工房メニューを開くだけで潜水艦情報を自動取得
- UI またはメモリからのデータソース切り替え対応
- キャラクター単位でのスナップショット管理

### 2. Overview ウィンドウ
- 全キャラクター・全潜水艦の状態を一覧表示
- 帰港予定時刻のソート・フィルタリング
- レスポンシブデザイン（3段階の列幅プリセット）
- コンパクト表示モード（残り時間短縮表示）

### 3. 通知システム
- Discord Webhook による通知（4隻分を集約）
- Notion Webhook による通知
- バッチング機能（指数バックオフリトライ付き）
- デッドレター管理と再送機能

### 4. 開発者ツール（DEV タブ）
- ForceNotifyUnderway トグル（開発モード）
- 手動送信ボタン
- トースト通知
- 送信ログ表示（最新10件）

## プロダクトの成功指標
1. **ユーザー満足度**: 潜水艦管理の手間削減を実感
2. **通知信頼性**: 通知成功率 95% 以上
3. **パフォーマンス**: UI 更新が 60fps を維持
4. **安定性**: クラッシュ率 < 0.1%

## 競合との差別化
- **自動化**: 手動操作不要でデータ取得
- **通知集約**: 4隻分をまとめて通知（通知スパム防止）
- **レスポンシブUI**: ウィンドウサイズに応じた最適表示
- **開発者ツール**: デバッグとテストを容易にする機能
